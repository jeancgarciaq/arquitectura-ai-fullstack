# 🏗️ Arquitectura AI: De Vibecoder a Ingeniero Fullstack

Bienvenido al repositorio oficial de **vibradev**. Este no es un curso de programación convencional; es un entrenamiento de alto nivel diseñado para dominar el desarrollo web Fullstack utilizando la **Inteligencia Artificial como mentor estratégico.**

## 🎯 El Concepto
A diferencia del *Vibecoding* tradicional (programar por instinto), aquí aplicamos **Ingeniería con Fundamento**. Utilizamos los hitos de [roadmap.sh](https://roadmap.sh/full-stack) para entender el "porqué" técnico antes de ejecutar un solo prompt.

## 🛠️ Metodología
- **IA como Tutor:** Utilizamos prompts diseñados para que la IA actúe como un Arquitecto Senior, explicando lógica y patrones de diseño.
- **Aprendizaje Flash:** Videos cortos, directos y enfocados en la acción.
- **Construcción Evolutiva:** Todo lo aprendido se aplica en tiempo real en nuestro proyecto principal: **Un CRM para Restaurante.**

## 📂 Estructura del Repositorio
- `/docs`: Aquí encontrarás los archivos `.md` con los **Prompts Maestros** de cada lección.
- `/backend`: Lógica de servidor construida con **Python**.
- `/frontend`: Interfaz de usuario interactiva con **JavaScript**.
- `/teoria`: Resúmenes de los fundamentos aprendidos según el roadmap.

## 🚀 Cómo participar
1. **Mira el video:** Busca el episodio correspondiente en [vibradev en YouTube](https://youtu.be/vq7pg3hhyVQ).
2. **Copia el Prompt:** Ve a la carpeta `/docs` y descarga el archivo `.md` de la lección.
3. **Ejecuta y Aprende:** Pega el prompt en tu IA (GEM/ChatGPT) y replica el código en tu editor.


## 🛠️ Requisitos Previos (Environment Setup)
Para seguir este curso y ejecutar el código de nuestro CRM para Restaurante, asegúrate de tener instalado lo siguiente en tu sistema (Windows, macOS o Linux):

- **Python (v3.10+):** Motor principal para nuestro backend y lógica de IA.

- **Node.js (v18+):** Necesario para la gestión de paquetes y ejecución del entorno frontend con JavaScript.

- **Git:** Para clonar este repositorio y gestionar tus propios avances.

- **Editor de Código:** Recomendamos Antigravity (o VS Code con extensiones de IA) para aprovechar la generación de código asistida.

- **Navegador Moderno:** Chrome, Brave o Firefox para las pruebas de interfaz.
